🚀 快速开始

欢迎使用 SecondMe API

SecondMe API 提供强大的数字分身能力，让你的应用能够访问用户的个人信息、软记忆，并以用户的 AI 分身进行智能对话。

👤
用户信息访问
获取用户基础信息、兴趣标签和个人知识库
💬
AI 对话

🧠
记忆管理

🔐 OAuth2 认证流程

SecondMe API 使用标准的 OAuth2 授权码流程 (Authorization Code Flow)，确保用户数据的安全访问。

完整流程可视化

1
用户访问应用
用户在你的应用中点击"使用 SecondMe 登录"
2
重定向到授权页
应用将用户重定向到 SecondMe 授权页面，携带 client_id、redirect_uri、state 参数
3
用户登录并授权
用户在 SecondMe 页面登录并确认授权你的应用访问其数据
4
返回授权码
SecondMe 重定向回你的 redirect_uri，URL 中包含授权码 (code) 和 state
5
授权码换 Token
应用后端使用授权码、client_secret 向 API 换取 Access Token 和 Refresh Token
6
调用 API
使用 Access Token 调用 SecondMe API，获取用户数据或进行对话

步骤 1: 构建授权 URL

JavaScript 
// 构建授权 URL
function buildAuthorizationUrl() {
  const params = new URLSearchParams({
    client_id: 'YOUR_CLIENT_ID',
    redirect_uri: 'https://your-app.com/callback',
    response_type: 'code',
    state: generateRandomState()  // 生成随机 state
  });
  return `https://go.second.me/oauth/?${params.toString()}`;
}

// 重定向到授权页
window.location.href = buildAuthorizationUrl();
⚠️ 安全提示
  • state 参数用于防止 CSRF 攻击，必须使用随机字符串
  • 收到回调后务必验证 state 参数与发起请求时一致
  • redirect_uri 必须使用 HTTPS（开发环境除外）

步骤 2: 授权码换 Token

POST /api/oauth/token/code
Bash 
curl -X POST "https://api.mindverse.com/gate/lab/api/oauth/token/code" \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "grant_type=authorization_code" \
  -d "code=YOUR_AUTHORIZATION_CODE" \
  -d "redirect_uri=https://your-app.com/callback" \
  -d "client_id=YOUR_CLIENT_ID" \
  -d "client_secret=YOUR_CLIENT_SECRET"
🚨 重要提示

必须使用 application/x-www-form-urlencoded 格式，不能使用 JSON 格式！

成功响应

{
  "code": 0,
  "data": {
    "accessToken": "lba_at_xxxxx...",
    "refreshToken": "lba_rt_xxxxx...",
    "tokenType": "Bearer",
    "expiresIn": 7200,
    "scope": ["user.info", "chat"]
  }
}

步骤 3: 刷新 Access Token

POST /api/oauth/token/refresh
Bash 
curl -X POST "https://api.mindverse.com/gate/lab/api/oauth/token/refresh" \
  -H "Content-Type: application/x-www-form-urlencoded" \
  -d "grant_type=refresh_token" \
  -d "refresh_token=YOUR_REFRESH_TOKEN" \
  -d "client_id=YOUR_CLIENT_ID" \
  -d "client_secret=YOUR_CLIENT_SECRET"
✅ 重要变更 (2026-03-11)

Refresh Token 不再轮换！返回的 refreshToken 与请求中的相同，可在 30 天有效期内重复使用。

🎫 令牌类型和有效期

令牌类型 前缀 有效期 用途
Authorization Code lba_ac_ 5 分钟 用于换取 Access Token，一次性使用
Access Token lba_at_ 2 小时 用于 API 调用，携带在请求头中
Refresh Token lba_rt_ 30 天 用于刷新 Access Token，可重复使用

权限 (Scopes)

权限 说明 分组
user.info 访问用户基础信息（姓名、邮箱、头像等） 用户信息
user.info.shades 访问用户兴趣标签 用户信息
user.info.softmemory 访问用户软记忆（个人知识库） 用户信息
chat 访问聊天功能 聊天
voice 使用语音合成功能 语音
💡 最佳实践

只请求必要的权限，避免请求过多权限导致用户拒绝授权。

⚠️ 错误处理

错误码命名规范

错误码遵循 {module}.{resource}.{reason} 格式：

  • module: 模块名称（oauth2、secondme 等）
  • resource: 资源类型（token、code、session 等）
  • reason: 错误原因（invalid、expired、not_found 等）

常见错误码

错误码 HTTP 状态码 说明
oauth2.token.expired 401 Access Token 已过期
oauth2.token.invalid 401 令牌无效
oauth2.scope.insufficient 403 权限不足
oauth2.code.expired 400 授权码已过期
oauth2.client.secret_mismatch 401 Client Secret 不匹配
secondme.session.not_found 404 会话不存在

错误处理最佳实践

检查业务状态码
首先检查响应体中的 code 字段判断请求是否成功（0 表示成功）
使用 subCode 进行程序化处理
根据 subCode 实现不同的错误处理逻辑
显示友好的错误信息
message 字段包含人类可读的错误描述，可直接展示给用户
实现重试策略
对于 5xx 错误，建议实现指数退避重试
Python 错误处理示例 
response = api_call()
data = response.json()

if data.get("code") == 0:
    # 请求成功
    process_data(data["data"])
else:
    # 请求失败
    sub_code = data.get("subCode")
    
    if sub_code == "oauth2.token.expired":
        # 刷新 token
        refresh_token()
    elif sub_code == "oauth2.scope.insufficient":
        # 提示用户权限不足
        show_permission_error()
    else:
        # 显示通用错误
        show_error(data.get("message"))

💻 代码示例

Python SDK 示例

Python 
import os
import requests
from urllib.parse import urlencode

class SecondMeClient:
    def __init__(self):
        self.client_id = os.getenv("SECONDME_CLIENT_ID")
        self.client_secret = os.getenv("SECONDME_CLIENT_SECRET")
        self.redirect_uri = os.getenv("SECONDME_REDIRECT_URI")
        self.base_url = "https://api.mindverse.com/gate/lab"
        self.access_token = None
    
    def get_authorization_url(self, state):
        """生成授权 URL"""
        params = {
            "client_id": self.client_id,
            "redirect_uri": self.redirect_uri,
            "response_type": "code",
            "state": state
        }
        return f"https://go.second.me/oauth/?{urlencode(params)}"
    
    def exchange_code_for_token(self, code):
        """授权码换 Token"""
        url = f"{self.base_url}/api/oauth/token/code"
        data = {
            "grant_type": "authorization_code",
            "code": code,
            "redirect_uri": self.redirect_uri,
            "client_id": self.client_id,
            "client_secret": self.client_secret
        }
        response = requests.post(url, data=data)
        result = response.json()
        if result["code"] == 0:
            self.access_token = result["data"]["accessToken"]
            return result["data"]
        raise Exception(f"Token exchange failed: {result}")
    
    def get_user_info(self):
        """获取用户信息"""
        url = f"{self.base_url}/api/secondme/user/info"
        headers = {"Authorization": f"Bearer {self.access_token}"}
        response = requests.get(url, headers=headers)
        return response.json()
    
    def chat_stream(self, message, **kwargs):
        """流式聊天"""
        url = f"{self.base_url}/api/secondme/chat/stream"
        headers = {
            "Authorization": f"Bearer {self.access_token}",
            "Content-Type": "application/json"
        }
        data = {"message": message, **kwargs}
        response = requests.post(url, headers=headers, json=data, stream=True)
        
        for line in response.iter_lines():
            if line:
                line = line.decode('utf-8')
                if line.startswith('data: '):
                    data = line[6:]
                    if data == '[DONE]':
                        break
                    yield data

JavaScript 示例

JavaScript 
// 环境变量配置
const config = {
  clientId: process.env.SECONDME_CLIENT_ID,
  clientSecret: process.env.SECONDME_CLIENT_SECRET,
  redirectUri: process.env.SECONDME_REDIRECT_URI,
  baseUrl: 'https://api.mindverse.com/gate/lab'
};

// 生成授权 URL
function getAuthorizationUrl(state) {
  const params = new URLSearchParams({
    client_id: config.clientId,
    redirect_uri: config.redirectUri,
    response_type: 'code',
    state: state
  });
  return `https://go.second.me/oauth/?${params.toString()}`;
}

// 授权码换 Token
async function exchangeCodeForToken(code) {
  const response = await fetch(`${config.baseUrl}/api/oauth/token/code`, {
    method: 'POST',
    headers: {
      'Content-Type': 'application/x-www-form-urlencoded'
    },
    body: new URLSearchParams({
      grant_type: 'authorization_code',
      code: code,
      redirect_uri: config.redirectUri,
      client_id: config.clientId,
      client_secret: config.clientSecret
    })
  });
  
  const result = await response.json();
  if (result.code === 0) {
    return result.data;
  }
  throw new Error(`Token exchange failed: ${result.message}`);
}

// 获取用户信息
async function getUserInfo(accessToken) {
  const response = await fetch(`${config.baseUrl}/api/secondme/user/info`, {
    headers: {
      'Authorization': `Bearer ${accessToken}`
    }
  });
  return await response.json();
}

// 流式聊天
async function chatStream(accessToken, message) {
  const response = await fetch(`${config.baseUrl}/api/secondme/chat/stream`, {
    method: 'POST',
    headers: {
      'Authorization': `Bearer ${accessToken}`,
      'Content-Type': 'application/json'
    },
    body: JSON.stringify({ message })
  });
  
  const reader = response.body.getReader();
  const decoder = new TextDecoder();
  
  while (true) {
    const { done, value } = await reader.read();
    if (done) break;
    
    const chunk = decoder.decode(value);
    const lines = chunk.split('\n');
    
    for (const line of lines) {
      if (line.startsWith('data: ')) {
        const data = line.slice(6);
        if (data === '[DONE]') return;
        console.log(data);
      }
    }
  }
}
